# About
---
CyberGuard is an educational mobile application that provides structured learning paths for **cybersecurity** concepts. The app combines theoretical knowledge with hands-on practice, allowing users to understand security principles through realistic scenarios and assessment tools.
## Features

Interactive Learning Modules - Structured curriculum covering fundamental to advanced cybersecurity topics
Real-World Scenarios - Simulated security threats and decision-based learning exercises
Quizzes and Assessments - Knowledge validation with detailed explanations and progress tracking
Core Topics - Network security, cryptography, authentication, social engineering, malware, web security, privacy, and incident response

## Installation
```
bash
git clone https://github.com/bakirmujicxu-dev/cybersec.git
cd cybersec
./gradlew assembleDebug
```
